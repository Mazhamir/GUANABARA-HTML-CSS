Pseudo-classes ( Definem elementos )

:hover: Aplica estilo quando o usuário passa o mouse sobre o
: visited: Aplica estilo a links que já foram visitados.
: active: Aplica estilo ao elemento no momento em que ele está sendo clicado.
: checked: Aplica estilo a elementos de formulário (como checkboxes ou radio buttons) que estão marcados.
: empty: Aplica estilo a elementos que não têm filhos (nem elementos nem texto).
:focus: Aplica estilo ao elemento quando ele recebe foco, como ao clicar em um campo de formulário.

Pseudo-elementos ( Definem conteúdos )

. :: before': Insere conteudo antes do elemento selecionado.
· :: after`: Insere conteúdo após o elemento selecionado.
. :: first-letter': Aplica estilo ao primeiro caractere do texto.
. :: first-line`: Aplica estilo à primeira linha do texto.
